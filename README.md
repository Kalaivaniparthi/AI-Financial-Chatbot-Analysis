# 🤖 GenAI Financial Chatbot & Data Analysis

## 📌 Project Overview
This repository contains my submission for the **Forage GenAI Job Simulation**. The project demonstrates:
- Manual extraction and analysis of financial data from SEC EDGAR 10-K filings.
- Building an AI-powered financial chatbot prototype that answers predefined queries using structured data.

**Certificate:** Completed June 2026

---

## 📊 Companies Analyzed
| Company | Ticker |
| :--- | :--- |
| **Microsoft** | MSFT |
| **Tesla** | TSLA |
| **Apple** | AAPL |

**Time Period:** Fiscal Years 2023, 2024, and 2025

---

## 📈 Metrics Extracted
The following five key financial figures were collected for each company and year:

| Metric | Source |
| :--- | :--- |
| **Total Revenue** | Income Statement |
| **Net Income** | Income Statement |
| **Total Assets** | Balance Sheet |
| **Total Liabilities** | Balance Sheet |
| **Operating Cash Flow** | Cash Flow Statement |

---

## 🤖 Chatbot Features
The chatbot responds to predefined financial queries, including:

| Query | Response |
| :--- | :--- |
| `"What companies are available?"` | Lists all companies in the database |
| `"Which company has the highest revenue in 2025?"` | Returns the top performer |
| `"How has [Company]'s revenue changed?"` | Shows revenue growth from 2024 to 2025 |
| `"How has [Company]'s net income changed?"` | Shows net income growth from 2024 to 2025 |
| `"Show summary for [Company]"` | Displays complete financial data |
| `"Show general summary"` | High-level overview of all companies |
| `"Show cash flow summary"` | Displays operating cash flow |
| `"help"` | Shows available commands |
| `"exit" or "quit"` | Ends the chat session |
